# 出張ラン向けコース拡充調査（大阪・福岡・仙台・名古屋）

調査日: 2026-07-12

## 目的

出張先として利用が多い大阪・福岡・仙台・名古屋について、主要駅・ホテル街から使いやすいランニングコースを優先して拡充する。都道府県の掲載数を均等に増やすのではなく、出張者が朝または夜に3〜7km走れることを重視する。

## 選定条件

- 主要駅・ホテル街から徒歩、または公共交通で短時間
- 3〜7kmを組み立てやすい（短い公式周回は複数周を想定）
- 初訪問でも迷いにくい周回・往復コース
- 信号が少ない、または公園・河川敷中心
- 自治体、観光局、公園管理者等の情報で場所・経路を確認できる
- 既存255スポットと同一コースではない
- 公開前にGPXを作成し、地図形状を目視確認できる

## 実装進捗

- `nakanoshima-loop`: BRouterで中之島外周6.78kmの周回GPXを作成。想定約6kmの±15%以内で、`gpx:check`の周回判定も合格したため登録データを作成。
- `sanno-park-fukuoka`: 初回生成1.97kmで公式1kmコースから大きく乖離。GPXは調査用として保存し、登録しない。
- `sendai-nishi-park`: 初回生成2.84kmで公式1kmコースから大きく乖離。GPXは調査用として保存し、登録しない。
- `higashi-park-fukuoka`: 初回生成1.09kmで公式760mコースから大きく乖離。GPXは調査用として保存し、登録しない。

## 既存スポットとの整理

- 福岡の舞鶴公園は大濠公園に隣接し、決まった周回コースもないため別スポットとしては追加しない。
- 名古屋城外周は既存の名城公園と近接するため追加しない。
- 仙台の広瀬川沿いは既存の「広瀬川河川敷（澱橋〜牛越橋）」と重複するため追加しない。
- 大阪城周辺は既存の大阪城公園でカバー済み。

## 根拠

- 福岡市公式観光情報「福岡のランニングコースおすすめ7選」: 山王公園は博多駅から約1kmで500m・1kmコース、東公園は760mの舗装周回、室見川は片道約4.7kmと案内されている。https://yokanavi.com/features/74419
- 仙台市「西公園北側エリア 1kmランニングコース」: 1周1km、平坦・舗装・信号なし、トイレ2か所と水飲み場あり。https://www.city.sendai.jp/aoba-chiikikatsudo/aobaku/machizukuri/ffi21.html
- 仙台市「青葉通のケヤキ並木」: 仙台駅から西公園・広瀬川・青葉山へ続く歩行空間として確認。https://www.city.sendai.jp/ryokuchihozen/kurashi/shizen/midori/reiwa100sen/aoba/aobadori.html
- 大阪市公園情報: 中之島公園は淀屋橋・北浜・なにわ橋から利用でき、業務地区の水辺に立地する。https://parksgreenery.city.osaka.lg.jp/article/parks-nakanoshima/
- 大阪市「大公園の魅力向上に向けた基本方針」: 中之島公園、靱公園、扇町公園が商業・業務系エリア、毛馬桜之宮公園が大川沿いの公園として整理されている。https://www.city.osaka.lg.jp/kensetsu/cmsfiles/contents/0000533/533506/honpen1.pdf

## 登録前チェック

1. 各候補の起点を主要駅側に置き、BRouterまたは実走GPXで代表経路を作る。
2. `npm run gpx:check` で距離・周回性・獲得標高を確認する。
3. 地図上で道路逸脱、橋の誤接続、往復スパーがないか目視する。
4. 公式情報でトイレ・給水・夜間照明を再確認し、不明値は推測しない。
5. 既存スポットから500m以内の場合は、同一場所の別コースになっていないか再確認する。
6. 合格した候補のみJSON化し、dry-run後に投入する。GPXが不確かな候補は公開しない。
