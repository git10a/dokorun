# 駅マスタの出典

- 出典: [駅データ.jp](https://ekidata.jp/)
- 収録版: 2024-04-26
- ファイル: `station.csv`, `line.csv`, `join.csv`

駅データ.jpの利用規約では、商用・非商用での利用、加工、無償での非加工データの第三者提供が認められています。出典表記は必須ではありませんが、更新時の確認先を残すためここに記載します。

今回は駅データ.jp由来の同版CSVを公開している Kaggle の `Japanese railway system` データセットから取得しました。年1回を目安に、駅データ.jpから取得した最新版CSVへ3ファイルをまとめて差し替え、`npm run station-data:build` を再実行してください。

無料版の当該CSVには `pref_cd` がないため、`postal-prefectures.json` は駅の郵便番号に対応する都道府県だけを日本郵便の全国郵便番号データ（2026-06-30版）から抽出した補完マスタです。`pref_cd` を含む駅CSVへ更新した後も、パイプラインはCSV側の値を優先します。
